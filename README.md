# Flowlytics AI - Business Intelligence Dashboard

[![Railway Deploy](https://railway.app/button.svg)](https://railway.app)

**Transform raw sales data into actionable business insights.**  
A SaaS platform built with n8n that analyzes e-commerce data to provide revenue reports, profit analysis, and growth recommendations.

##  Live Demo
**URL:** `https://your-app.railway.app`  
**Login:** admin / (your password)

##  Features
-  **Revenue & Profit Analytics** - Real-time financial dashboard
-  **Top Product Identification** - Find your best-selling items
-  **City-wise Performance** - Geographic revenue analysis
-  **Payment Method Insights** - Optimize payment options
-  **Risk Detection** - Identify high cancellation areas
-  **Growth Recommendations** - AI-powered business advice

## Tech Stack
- **Automation:** n8n
- **Database:** PostgreSQL
- **Hosting:** Railway.app
- **Authentication:** n8n Basic Auth
